How to use — read once, you're done
☀️ Every morning
  • Hit Pull Orders — Shopify loads automatically
  • Check Bagging tab for today's list
  • Check Roasting tab for green coffee and batch counts
  • Start here before printing Shopify labels
📦 Bagging tab
  • Needed = total bags required
  • Bagged = tap + as you bag each one
  • Remaining = what's still left to bag
  • Row auto-completes when Bagged = Needed
  • Lock a row to protect qty from Shopify refresh
  • Sort by bag size to group work efficiently
🔥 Roasting tab
  • Batches Needed = calculated from remaining bags
  • Tap + on Batches Done as you finish each drop
  • Remaining updates live across all devices
  • Red banner = broken blend ratio — fix in Blends tab
➕ Add Order
  • For café, wholesale, or staff orders not in Shopify
  • Pick coffee → bag size → qty → Add to Plan
  • Shows instantly in Bagging and Roasting
⚗️ Blends tab
  • Edit blend recipes — ratios must add to 100%
  • All coffees pulled from Shopify — no free typing
  • Red border = blend has a retired coffee — update it
  • Edit single origin batch sizes here too
🌙 End of day
  • Completed rows auto-mark Done when fully bagged
  • Click Archive Done to clear completed rows
  • View Archive shows what was completed today
  • Partially done rows carry over to tomorrow automatically
Pull Orders anytime — locked rows won't be overwritten. Unlocked rows recalculate from current Shopify qty. All changes sync instantly across all devices.
