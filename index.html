How to use this tool — read once, you're done
☀️ Every morning
  • Hit Pull Orders — Shopify loads automatically
  • Check the Bagging tab for today's list
  • Check the Roasting tab for green coffee and batch counts
📦 Bagging tab
  • Check ✓ Done as each product is bagged
  • Edit qty if an order changes mid-day
  • Hit Lock to protect a row from Shopify refresh overwriting it
  • Use the Sort By dropdown to group by bag size or type
  • Internal = orders added manually by the team
🔥 Roasting tab
  • Shows exact green lbs and batch count per coffee
  • Check ✓ Done per row as you finish each roast
  • Add one-time custom roasts with the + Add button
  • Red banner = a blend ratio is broken — fix it in Blends tab
➕ Add Order tab
  • For café, wholesale, or staff orders not in Shopify
  • Pick coffee → bag size → qty → Add to Plan
  • Shows up instantly in Bagging and Roasting
⚗️ Blends tab
  • Edit blend recipes — ratios must add to 100%
  • Edit single origin batch sizes here
  • Changes take effect on Roasting plan immediately
🌙 End of day
  • Check all done rows ✓
  • Click Archive Done — removes them from the active plan
  • Click View Today's Archive to see what was completed
  • Slate is clean for tomorrow
Pull Orders anytime throughout the day — locked rows won't be overwritten. Unlocked rows update to match current Shopify qty. All changes sync instantly across all devices.
