How to use — read once, you're done
☀️ Every morning — start here
  • Hit Pull Orders — loads all unfulfilled Shopify orders
  • Subscription orders auto-resolve to this week's coffee from the Subscriptions tab
  • Check Bagging tab for today's pick/pack list
  • Check Roasting tab for green coffee and batch counts
  • Do this before printing Shopify labels
📦 Bagging tab
  • Needed = total bags required from all orders
  • Bagged = tap + each time you bag one
  • Remaining = bags still to go
  • Row auto-completes when Bagged = Needed
  • Lock protects a row from being overwritten on next Pull
  • Sort by bag size or type to match your physical workflow
  • Oldest order date shown — amber = order is from a prior day
  • Internal tag = order added manually via Add Order tab
🔥 Roasting tab
  • Driven by Remaining bags — updates as you bag
  • Batches Needed = green lbs ÷ batch size (15% shrinkage applied)
  • Tap + on Batches Done as you finish each drop
  • Remaining batches updates live across all devices
  • Lock freezes a row so new orders don't change it mid-roast
  • Add custom one-time roasts with the + Add button
  • Red banner = broken blend ratio or retired coffee — fix in Blends tab
📅 Subscriptions tab
  • 13-week rolling schedule — 6 past (locked), current (amber), 6 future
  • Set this week's coffee for Modernist, Classicist, Espressoist
  • Pull Orders uses this to replace subscription line items with real coffee names
  • Past weeks are permanent history — no editing
  • Use + Add Future Week to extend the schedule
  • Save button per row — changes don't save until you click Save
⚖️ Mix Guide tab
  • Full-screen blend reference — big bold percentages for bagging station
  • Tap a blend name at the top to switch between drops
  • Bag Calculator above the card — enter qty of each bag size
  • Calculator outputs exact lbs and grams of each component coffee to weigh
  • Resets automatically when you switch blends
  • No login, no persistence — real-time calculator only
🏢 B2B tab
  • Shows all wholesale orders grouped by company name
  • Double-check against Shopify for multi-order accounts
  • Read-only — for reference and shift handoff
  • Hit Refresh B2B to reload latest
🚚 Ship tab — buy USPS labels
  • Lists each B2B order with its ship-to and how many 14×10×10 boxes it packs into
  • Tap Ship → to open an order; the box split is auto-figured — eyeball it
  • Get rates shows the cubic Ground Advantage cost per box — no money moves yet
  • Buy labels charges Shippo only after you confirm the total
  • If a box fails, only the failed ones retry — bought labels are never re-charged
  • Open label opens the Shippo PDF in a new tab — print it yourself and pick the Rollo; Print slip auto-opens the print dialog for the contents slip (first time, you may need to pick 4×6 paper size manually in Safari)
  • Fulfill order writes all tracking to Shopify with one customer email — only after every box has a label
  • An already-fulfilled order shows calmly and is never notified twice
➕ Add Order
  • For café, wholesale, or staff orders not in Shopify
  • Pick coffee → bag size → qty → note → Add to Plan
  • Shows instantly in Bagging and Roasting tabs
  • Tagged as "Internal" so it's visually distinct from Shopify orders
⚗️ Blends tab
  • Edit blend recipes — ratios must add to 100% or roasting math breaks
  • Coffee dropdowns sourced from Shopify active products only
  • Red border on a blend card = retired coffee in that blend — update it
  • Single origin batch sizes editable here — new coffees default to 8lb
  • Decaf coffees = 7lb batch size
🌙 End of day
  • Rows auto-complete when Bagged = Needed
  • Click Archive Done to sweep completed rows out of the plan
  • View Archive shows everything completed today — useful for shift handoff
  • Partial rows carry forward automatically — nothing is lost
  • Roast batch counts reset at midnight for the next day
🔄 How Pull Orders works
  • Recalculates Needed from scratch based on current Shopify unfulfilled orders
  • Preserves Bagged counts — never overwrites work already done
  • Locked rows are never touched by a pull
  • Cancelled or fulfilled orders are automatically removed
  • New active coffees in Shopify are auto-added to settings at 8lb default
  • Retired coffees are auto-removed from settings
All changes sync in real time across every device. Pull Orders anytime — locked rows are never overwritten. Partial work carries forward. History lives in View Archive.
